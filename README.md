## Usage
```
input > {検索ワード}
output > {検索ワード}の要約と関連ワードを表示します。

input > :setup_lang=en
output > wikipediaの検索結果を任意の言語に変更します。

input > :history
output > 最大13件の検索履歴を表示します。
```
